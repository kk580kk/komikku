## 🎯 测试功能清单

### 1️⃣ CBZ 封面提取测试

**测试目的**: 验证从 CBZ 压缩包自动提取封面功能

**测试步骤**:
```
1. 在平板上准备测试漫画文件夹
   路径示例：/storage/emulated/0/Download/E-Hentai (ZH)/TestManga/
   
2. 放入 CBZ 文件
   - chapter1.cbz (包含多张图片)
   - 确保第一张图片是封面
   
3. 打开 Komikku 应用
   设置 → 高级 → 导入本地下载
   
4. 勾选选项:
   ☑️ 添加到默认分类
   ☐ 后台导入 (先测试前台)
   
5. 点击"导入"
   
6. 检查结果:
   - 漫画是否出现在图书馆
   - 封面是否正常显示
   - 点击漫画能否看到章节
```

**预期结果**:
- ✅ 漫画成功导入图书馆
- ✅ 封面显示为 CBZ 中的第一张图片
- ✅ 章节列表正常显示

**常见问题**:
- 封面不显示 → 检查 CBZ 是否包含图片
- 章节不显示 → 检查 CBZ 文件格式

### 2️⃣ 后台导入通知测试

**测试目的**: 验证后台导入完成后通知栏显示

**测试步骤**:
```
1. 准备多个漫画文件夹（建议 10+ 个）
   
2. 打开 Komikku 应用
   设置 → 高级 → 导入本地下载
   
3. 勾选选项:
   ☑️ 添加到默认分类
   ☑️ 后台导入 ⭐
   
4. 点击"导入"
   
5. 关闭导入对话框
   
6. 下拉通知栏查看:
   - 是否显示导入进度
   - 完成后是否显示结果通知
   
7. 点击通知:
   - 是否返回应用
```

**预期通知内容**:
```
本地导入
导入：15 | 跳过：3 | 错误：0
```

**检查点**:
- [ ] 通知栏显示导入进度
- [ ] 完成后显示结果
- [ ] 点击通知返回应用
- [ ] 通知可以滑动删除

### 3️⃣ 自动导入搜索优化测试

**测试目的**: 验证关键词搜索和匹配功能

**测试步骤**:
```
1. 打开 Komikku 应用
   更多 → 自动导入
   
2. 输入测试标题（使用真实漫画名）:
   示例 1: "[C89] [Artist] Manga Title (Chinese)"
   示例 2: "One Piece"
   示例 3: "Manga Series [Translator]"
   
3. 点击"扫描本地文件夹"（可选）
   自动填充已下载的漫画名
   
4. 点击"开始导入"
   
5. 观察搜索过程:
   - 是否尝试完整标题搜索
   - 失败后是否使用关键词搜索
   - 最终是否成功匹配
```

**测试用例**:

| 输入标题 | 预期关键词 | 期望结果 |
|---------|-----------|---------|
| `[Author] Manga Title` | `Author`, `Manga Title` | 匹配成功 |
| `Manga (Subtitle) [Artist]` | `Artist`, `Manga`, `Subtitle` | 匹配成功 |
| `One Piece` | `One Piece` | 直接匹配 |

**检查点**:
- [ ] 完整标题优先搜索
- [ ] 失败后使用关键词
- [ ] 中括号内容被提取
- [ ] 相似度 ≥ 40% 才导入
- [ ] 显示导入结果统计
