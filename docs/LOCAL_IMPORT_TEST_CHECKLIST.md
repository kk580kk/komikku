## 使用说明摘要

### 手动导入（本地下载目录 → 书架）
1. **设置** → **高级** → **导入本地下载**
2. 选择是否「添加到默认分类」、是否「后台导入」
3. 点击 **导入**，等待完成或关闭对话框（后台模式）后在通知栏查看结果

### 自动导入（标题 → 搜索匹配 → 导入）
1. **更多** → **自动导入**
2. 在输入框中填写漫画标题（一行一个，支持 `[作者]`、`(副标题)` 等）
3. 可选：**扫描本地文件夹** 再 **开始导入**
4. 查看列表中的匹配与导入结果，必要时在通知栏查看汇总

### 结果通知格式
- **本地导入**：`导入：15 | 跳过：3 | 错误：0`
- **自动导入**：以实际实现为准（如导入数、失败数等）
