## 🔧 技术细节

### 自适应并发策略
```kotlin
private fun calculateAdaptiveConcurrency(total: Int): Int {
    return when {
        total < 20 -> 5   // 小批量：快速完成
        total < 50 -> 3   // 中批量：平衡速度和稳定性
        else -> 2         // 大批量：避免触发源限制
    }
}
```

### 通知更新策略
- **开始通知**: 立即显示，包含总项目数
- **进度通知**: 每完成 10 个项目更新一次
  - 显示：当前进度 / 总数 / 已成功 / 已失败
- **完成通知**: 导入完成后显示最终统计

### 标题提取逻辑
```kotlin
private fun extractTitleFromPath(path: String): String {
    return path
        .substringAfterLast('/')
        .substringAfterLast('\\')
        .trim()
}
```
